# Getting started

To make the in-game server browser work and to get new game features 
you need to patch your game.

Pick instructions appropriate for you:

<div class="grid cards" markdown>

-   :material-gamepad-variant:{ .lg .middle .accent } __Players__

    ---

    You want to play the game with working multiplayer and get additional features.

    [:octicons-arrow-right-24: Installation](players.md)

-   :fontawesome-solid-server:{ .lg .middle .accent } __Server owners__

    ---

    You want to host a server that others can join, get additional features and be protected against common exploits.

    [:octicons-arrow-right-24: Installation](server-owners.md)

-   :material-hammer-screwdriver:{ .lg .middle .accent } __Manual Online Patching Tool__

    ---

    You have all the required files and you only want to patch the game on your own to have working in-game server browser.

    [:octicons-arrow-right-24: Instructions](manual-patching.md)

-   :material-folder-multiple:{ .lg .middle .accent } __Other__

    ---

    If you need other versions of binaries that are pre-patched.

    [:octicons-arrow-right-24: Browse](other.md)

</div>
